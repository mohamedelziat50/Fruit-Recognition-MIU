This project is a fruit image classifier built with machine learning and image processing techniques. The website UI is designed to let users upload fruit images for real-time prediction and classification.

Dataset Fruit Categories (230 Classes)

Project Section

This project uses a simple classical computer-vision approach. The fruit image is processed by converting to the saturation channel (S-channel) of the HSV color space. A binary mask is created using thresholding, which isolates the fruit region. This segmented mask is then used to extract features for classification.

Dataset Reference: Fruits-360 Dataset
